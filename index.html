2分钟搭建属于自己的AI生图网站
日常 2025/8/23

原创

项目地址：https://github.com/zhumengkang/cf-ai-image

注意：worker.js文件的第75行是密码，默认密码是admin123
大家可以自行修改

本期文案：

大家好，今天教大家用cloudflare搭建一个在线文生图的网站
首先给搭建演示一下这个网站，来到这里输入默认密码admin123
这里咱们来个随机提示词，图片数量选择8，直接生成
等个几秒就行，这个项目的作者说这个ai的性能比5080还强
好了，这就是这个项目的github地址，大家可以给作者点个star
这里我就folk一下，其实大家可以不用folk
接下来咱们来到cf，点works，创建一个Helloword，名字随便起
然后点编辑代码，咱们去Github打开worker这个文件，复制带码
回到cf粘贴，再新建一个index.html，再去Github上的index文件复制代码
好，咱们回到worker文件，找到第75行，这里是密码，默认是admin23，咱们可以改一下
这里我就改成了B站的uid，然后点击部署
接下来咱们回到workers，点击绑定，选择Workers AI，这里变量名称必须是大写的AI
然后咱们再访问一下，这里随便生成一下
这个项目是支持批量下载图片的，咱们下载一下压缩包，可以查看，没有问题
后续大家可以绑定上自定义域名，本期视频电赞破50，教大家美化这个页面

日常 2025/7/22

原创

今天我带来的教程绝对好用

需要用的代码：

一键换软件源命令

bash <(curl -sSL https://linuxmirrors.cn/main.sh)
点击复制命令

一键docker换源

bash <(curl -sSL https://linuxmirrors.cn/docker.sh)
点击复制命令

使用中国区Azure源安装Docker

curl -fsSL https://get.docker.com | bash -s docker --mirror AzureChinaCloud
点击复制命令

请使用以下命令来下载安装脚本：

curl -fsSL https://get.casaos.io -o casaos_install.sh
点击复制命令

下载完成后，您就可以使用文本编辑器（如 nano 或 vi）打开 casaos_install.sh 文件进行编辑了：

nano casaos_install.sh
点击复制命令

在脚本第104行的https://github.com/改为以下代码

https://github.bigjackson.top/github.com/
点击复制命令

编辑完成后，保存文件，然后您可以手动执行这个修改后的脚本：

sudo bash casaos_install.sh
