2026年最新免费无限制域名邮箱搭建教程
日常 2026/1/10

本期视频将教大家用GitHub上的开源项目cloud-mail来搭建一个永久免费的域名邮箱

域名邮箱的优点 ✅

专业形象 ✉️：使用公司或个人域名（如 tud8951@bigjackson.top）比通用免费邮箱更显可信和正规。
品牌一致性 🔖：邮箱与网站、名片、宣传材料保持统一，强化品牌识别。
可控性与自主管理 ⚙️：可创建/删除帐号、别名和转发规则，灵活满足组织需求。
更高的安全性 🔒：可配置 SPF/DKIM/DMARC、强制 MFA 和企业级安全策略，降低被滥用或被冒用风险。
更好送达率 📬：正确配置后减少被识别为垃圾邮件的概率，提高邮件到达率。
合规与归档能力 🗂️：便于实施邮件备份、审计和合规策略，满足法规或公司政策要求。
长期稳定性 ⏳：不依赖第三方免费邮箱政策变化，域名和服务可由自己或受信任的供应商长期维护。
集成与自动化 🔧：方便与自建系统、CRM、工单、API 等集成，支持自动化工作流。

workers路径：/mail-worker

环境变量：

admin 管理员邮箱，格式为xxx@xxx.xxx
domain 域名，格式为[“domain1.com”,”domain2.com”]
jwt_secret 随机字符，谁便打几十个字母和数字就行

绑定数据库：
KV 命名空间 = kv
D1 数据库 = db
R2 存储桶 = r2

初始化：https://你绑定的域名/api/init/你环境变量里设置的随机字符

resend webhooks：https://你的域名/api/webhooks

注意：一定要取消勾选非生产分支

注意：目前项目官网已经下架cloudflare workers部署教程
