日常 2025/7/22

原创

今天教大家用OECT搭建CasaOS，虽然别的大佬已经发过教程了，但是他们的教程大陆用不了

今天我带来的教程绝对好用

需要用的代码：

一键换软件源命令

bash <(curl -sSL https://linuxmirrors.cn/main.sh)
点击复制命令

一键docker换源

bash <(curl -sSL https://linuxmirrors.cn/docker.sh)
点击复制命令

使用中国区Azure源安装Docker

curl -fsSL https://get.docker.com | bash -s docker --mirror AzureChinaCloud
点击复制命令

请使用以下命令来下载安装脚本：

curl -fsSL https://get.casaos.io -o casaos_install.sh
点击复制命令

下载完成后，您就可以使用文本编辑器（如 nano 或 vi）打开 casaos_install.sh 文件进行编辑了：

nano casaos_install.sh
点击复制命令

在脚本第104行的https://github.com/改为以下代码

https://github.bigjackson.top/github.com/
点击复制命令

编辑完成后，保存文件，然后您可以手动执行这个修改后的脚本：

sudo bash casaos_install.sh
点击复制命令
